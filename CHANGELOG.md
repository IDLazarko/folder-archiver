# История изменений

## [2.0.0] - 2024-01-15
### Добавлено
- Поиск папок второго уровня вложенности
- Прогресс-бары для отслеживания процесса
- Предварительный просмотр операции
- Автоматическая очистка пустых родительских папок
- Читаемое форматирование размеров файлов
- Детальная статистика сжатия

### Изменено
- Именование архивов теперь включает родительскую папку
- Улучшена обработка ошибок
- Оптимизировано использование памяти

## [1.0.0] - 2024-01-01
### Добавлено
- Базовая архивация папок первого уровня
- Поддержка ZIP формата
- Удаление оригинальных папок
